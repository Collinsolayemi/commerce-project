# online-buying-and-selling


## Description

### A `Node.JS` _functional project that was built along with_ `ExpressJS` and `MongoDB` _for the database_. In this project a user can buy and sell any product of thier choice .




## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
